## Libraries & Dependencies

- **React Native Paper** — UI component library
- **@expo/vector-icons** — icons
- **react-native-gifted-charts** — chart generation
- **react-native-svg** — SVG support for charts
- **react-native-linear-gradient** — gradient dependency for charts
- **Jest** — unit testing
- **Drizzle ORM** — database ORM
- **dayjs** — date/time range calculations
- **@react-native-async-storage/async-storage** — dark/light mode toggle persistence
- **expo-file-system & expo-sharing** — file export support
- **papaparse** — export applications table as CSV/spreadsheet
- **expo-notifications** — push notifications
- **bcryptjs** — password hashing
